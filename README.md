# disegni-studio

אתר סטטי (HTML/CSS) עבור disegni.studio, עם ניהול תוכן דרך Decap CMS.

## מבנה
- `css/` — עיצוב
- `js/` — סקריפטים
- `images/` — תמונות האתר
- `admin/` — ממשק Decap CMS
- `content/` — קבצי תוכן (טקסטים) שה-CMS עורך
- `reference/` — גיבויי SingleFile של האתר הישן (רפרנס בלבד, לא חלק מהאתר, לא ב-git)

## פרסום
git push ל-GitHub → cPanel Git Version Control מושך אוטומטית ל-Ultahost.
